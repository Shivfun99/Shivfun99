Adaptive Linux Memory Governor written in C++.

Instead of waiting until memory is exhausted, Memopt++ proactively reduces memory pressure using:

- cgroups v2
- ZRAM Compression
- KSM Deduplication

### Tech

`C++17` • `Linux` • `Memory Management` • `System Programming`
